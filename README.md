# AI Security Research

## Overview

This repository contains research notes, security concepts, and analysis focused on modern AI systems, Large Language Models (LLMs), AI-assisted workflows, and emerging attack surfaces in AI-integrated environments.

The goal of this repository is to explore how modern AI systems interact with APIs, authentication mechanisms, external services, and trust boundaries from an offensive and defensive security perspective.

---

## Research Topics

### Prompt Injection
- Indirect prompt injection
- Context manipulation
- Instruction override techniques
- Multi-step prompt chaining risks

### AI SSRF Behaviors
- Server-side request forgery risks in AI systems
- Internal fetch mechanisms
- External URL processing
- AI-assisted request handling

### Authentication & Identity
- Authentication context handling
- Session propagation risks
- OAuth and token-related observations
- Identity boundary considerations

### API Security
- AI interaction with external APIs
- GraphQL attack surface observations
- API trust boundaries
- Improper authorization concepts

### AI System Attack Surfaces
- Plugin ecosystems
- Agent workflows
- Tool invocation risks
- Data exposure scenarios
