# Prompt Injection Notes

## Overview

Prompt injection occurs when attacker-controlled input manipulates the behavior or instructions of a Large Language Model (LLM).

## Key Risks

- Instruction override
- Data exfiltration
- Indirect prompt injection
- Multi-step prompt chaining
- Trust boundary confusion

## Security Considerations

- Input validation
- Context separation
- Tool access restrictions
- Output filtering
- Least privilege design

## Areas for Future Research

- AI agent workflows
- Tool invocation security
- Multi-agent trust boundaries
